# Xdebug Docker
Container to using xdebug on php in docker.

## Dependencies
- Docker

## Running

1. Download project
```bash
git clone https://github.com/luisaugustomelo/xdebug-docker.git <NEW_NAME>

cd <NEW_NAME>
```

2. Run docker-compose command
```bash
docker-compose up
```

3. Listing images
```bash
docker images ls
```
